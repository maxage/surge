# 🌊 Surge 高级定制配置文件

欢迎使用这份由我和 Gemini Pro 共同打造的 Surge 深度定制配置文件！🎉

这份配置的目标是实现**高度自动化**、**极致灵活性**和**清晰可维护性**的完美结合。
无论您是想稳定访问外部网络、解锁流媒体，还是想对特定应用进行精细的流量控制，它都能满足您的需求。

---

## 核心特性 ✨

* **智选 + 手选 + 故障转移 (`故转`)**: 融合了 `select`, `url-test` 和 `fallback` 的优点。您可以为常用地区**固定一个节点**以保持 IP 稳定，当该节点失效时，系统会**自动切换**到延迟最低的备用节点，待手动节点恢复后又**自动切回**。兼顾稳定与智能！
* **区域化节点分组**: 自动将您的机场节点按地区（香港、美国、日本、台湾、韩国、新加坡等）及特殊用途（低价区、欧洲等）进行分类，一目了然。
* **精细化应用分流**: 为常用 App（如 GitHub, AI服务, Netflix, YouTube 等）创建了独立的策略组，可以单独控制每个 App 的出口策略，互不干扰。
* **强大的广告拦截与隐私保护**: 集成了社区优选的拦截规则，可以有效屏蔽广告、追踪、病毒及钓鱼网站，并优化了特定 App 的网络体验。
* **智能兜底策略**:
    * **`🐟 漏网之鱼`**: 所有未被精确匹配到的流量，都会被导向此策略组。您可以通过手动选择它的出口，来应对各种临时的、未知的网络访问需求。
    * **`🪄 手动选择`**: 这是一个“总开关”，可以一键改变大多数应用的全局出口，方便快捷。

---

## 策略组设计理念 🧠

本配置的策略组经过精心设计，形成了一个清晰的四层依赖结构：

1.  **[底层] 节点池 🏊‍♂️**:
    * `✈️ 主力机场` / `🚀 备用机场`: 您的原始订阅链接。
    * `🇭🇰 香港-手动`: 从订阅中筛选出的香港地区节点，供您手动选择。
    * `🇭🇰 香港-智选`: 自动测试延迟最低的香港节点。

2.  **[逻辑层] 故转核心 🔄**:
    * `🇭🇰 香港-故转`: 融合了“手动”与“智选”。优先使用您手动选择的节点，失效时自动切换到“智选”节点。这是我们这套配置的精髓！

3.  **[控制层] 核心开关 🕹️**:
    * `🪄 手动选择`: 您的全局“遥控器”，可以选择让大部分应用通过哪个区域的 `-故转` 组出口。
    * `🐟 漏网之鱼`: 未知流量的“收容所”，您可以决定这些流量的最终去向。

4.  **[应用层] 具体服务 📱**:
    * `🤖 人工智能`, `🎬 网飞视频`, `💻 GitHub`... 每个策略组都像一个独立的 App 开关。您可以让它跟随“手动选择”的全局设定，也可以为它单独指定一个出口（比如强制“网飞视频”走“日本-故转”）。

---

## 如何使用 🚀

1.  **下载配置文件**: 将本仓库中的 `.conf` 文件下载到您的设备。
2.  **填入订阅链接**:
    * 用文本编辑器打开 `.conf` 文件。
    * 找到 `[Proxy Group]` 部分的 `✈️ 主力机场` 和 `🚀 备用机场`。
    * 将 `policy-path=` 后面的 `#####订阅链接#####` 替换为您自己的机场订阅链接。
3.  **导入 Surge**:
    * **macOS**: 将修改后的 `.conf` 文件拖入 Surge 窗口，或从菜单中选择“从URL导入”。
    * **iOS**: 通过 AirDrop、iCloud Drive 或其他方式将文件导入，然后在 Surge App 中选择该配置。
4.  **开始使用**:
    * 在 Surge 的策略组菜单中，找到 `🪄 手动选择`，选择您偏好的默认出口区域（例如 `🇭🇰 香港-故转`）。
    * 对于有特殊需求的 App（如 `🎬 网飞视频`），可以单独进入其策略组，选择指定的区域出口。
    * 对于未知的网站访问（如土耳其礼品卡网站），可以在 `🐟 漏网之鱼` 策略组中选择 `🛍️ 低价购买-故转` 来临时改变出口。
